## About This Project

This is my personal portfolio website built with Next.js 16, featuring a modern design and optimized performance. The site showcases my projects, skills, and professional experience in web development.

## Technical Stack

The website is built using cutting-edge technologies to ensure optimal performance and user experience:

- **Next.js 16** with App Router for modern React development
- **TypeScript** for type-safe code
- **Tailwind CSS** for styling with a custom grayscale design system
- **GSAP** for smooth animations
- **next-intl** for internationalization
- **MDX** for content management

## Key Features

- Modern and responsive design
- Optimized performance with Server Components
- SEO friendly architecture
- Dark mode support
- Internationalization (English/Spanish)
- Smooth animations and transitions

## Implementation Details

The portfolio follows best practices for modern web development, including:

- Server-side rendering for optimal performance
- Static generation where possible
- Optimized images with Next.js Image component
- Accessibility-first approach
- Mobile-first responsive design
