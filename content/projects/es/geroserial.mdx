## Sobre Este Proyecto

Este es mi sitio web de portfolio personal construido con Next.js 16, con un diseño moderno y rendimiento optimizado. El sitio muestra mis proyectos, habilidades y experiencia profesional en desarrollo web.

## Stack Técnico

El sitio web está construido utilizando tecnologías de vanguardia para garantizar un rendimiento y experiencia de usuario óptimos:

- **Next.js 16** con App Router para desarrollo moderno con React
- **TypeScript** para código con tipos seguros
- **Tailwind CSS** para estilos con un sistema de diseño grayscale personalizado
- **GSAP** para animaciones suaves
- **next-intl** para internacionalización
- **MDX** para gestión de contenido

## Características Principales

- Diseño moderno y responsive
- Rendimiento optimizado con Server Components
- Arquitectura amigable para SEO
- Soporte para modo oscuro
- Internacionalización (Inglés/Español)
- Animaciones y transiciones suaves

## Detalles de Implementación

El portfolio sigue las mejores prácticas para el desarrollo web moderno, incluyendo:

- Renderizado del lado del servidor para rendimiento óptimo
- Generación estática cuando es posible
- Imágenes optimizadas con el componente Image de Next.js
- Enfoque en accesibilidad
- Diseño responsive mobile-first
